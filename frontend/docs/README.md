# Sand Table Control Application - Documentation

Welcome to the Sand Table Control Application documentation. This application provides a modern web interface for controlling kinetic sand tables driven by Arduino/ESP32 microcontrollers.

## Quick Start

1. Configure your controller connection in Settings
2. Browse patterns in the Pattern Gallery
3. Select a pattern to view controls and preview
4. Use the LED Control page to customize lighting
5. Monitor machine status in the Dashboard

## Features

- **Pattern Control**: Browse, select, and run predefined patterns
- **Custom Creation**: Create patterns via sketch, image upload, or text prompts
- **LED Control**: Full RGB lighting control with presets
- **Live Dashboard**: Real-time machine status monitoring
- **Safety Features**: Emergency stop, confirmations, auto-resume
